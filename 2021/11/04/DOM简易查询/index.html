06.1 Node

Node接口属性

写入时 读取时
node.innerHTML 转化标签 转化标签
node.textContent 不转化标签 不转化标签
  • Node.nodeType –返回节点类型
  • Node.nodeName –返回节点名称
  • Node.nodeValue –可读写，返回文本值，只用于文本节点
  • Node.textContent–如上表格
  • Node.baseURI–返回文档地址
  • Node.ownerDocument–返回顶层文档对象，document
  • Node.nextSibling–下一个同级节点
  • Node.previousSibling–上一个
  • Node.parentNode–父节点（元素、文档、文档片段）
  • Node.parentElement–父元素节点
  • Node.firstChild，Node.lastChild–子节点
  • Node.childNodes–子节点集合
  • Node.isConnected–返回布尔值，当前节点是否在文档中

Node接口

  • Node.appendChild()–末端插入
  • Node.hasChildNodes()–布尔值，是否有子节点
  • Node.cloneNode()–克隆节点
  • Node.insertBefore()–插入节点
  • Node.removeChild()–移除节点
  • Node.replaceChild()–置换节点
  • Node.contains()–是否包含某节点
  • Node.compareDocumentPosition()–比较节点位置
  • Node.isEqualNode()，Node.isSameNode()–判断相似&相同
  • Node.normalize()–清理内部文本节点
  • Node.getRootNode()–返回当前文档根节点

NodeList接口

  • NodeList.prototype.length–长度
  • NodeList.prototype.forEach()–遍历
  • NodeList.prototype.item()–返回某位置的成员
  • NodeList.prototype.keys()，NodeList.prototype.values()，NodeList.prototype.entries()–

6.2 Document

文档静态属性

document.links–返回当前文档所有设定了href属性的<a>及<area>节点

document.documentURL–继承自Document接口，可用于所有文档

document.URL–继承自HTMLDocument接口，只能用于 HTML 文档

document.domain–返回当前文档的域名，不包含协议和接口

document.location–

document.lastModified–文档最后修改时间

document.title–文档标题，可写

document.characterSet–返回当前文档的编码

document.referrer–表示当前文档的访问者来自哪里

document.dir–文字方向

document.compatMode–返回浏览器处理文档的模式

文档状态属性

document.hidden–页面是否可见，一般配合Page Visibility API

document.visibilityState–返回文档的可见状态

document.readyState–返回当前文档的状态

其他属性

document.cookie–操作浏览器 Cookie

document.designMode–控制当前文档是否可编辑

document.implementation–创建独立于当前文档的新DocumentType对象

方法

document.elementFromPoint(x,y)–返回指定位置最上层元素节点

document.elementsFromPoint(x,y)–返回指定位置元素节点组

document.creatElement()–生成元素节点

document.creatTextNode()–生成文本节点

document.creatAttribute()–生成属性节点

document.createComment()–生成注释节点

document.createDocumentFragment()–生成空的文档片段节点

document.addEventListener()，document.removeEventListener()，document.dispatchEvent()–触发事件

document.hasFocus-返回一个布尔值，表示当前文档之中是否有元素被激活或获得焦点。

6.5 Text节点和DocumentFragment节点

属性

data == nodeValue – 获取文本内容，可设置

length – 文本长度

方法

  • appendData()：在Text节点尾部追加字符串。
  • deleteData()：删除Text节点内部的子字符串，第一个参数为子字符串开始位置，第二个参数为子字符串长度。
  • insertData()：在Text节点插入字符串，第一个参数为插入位置，第二个参数为插入的子字符串。
  • replaceData()：用于替换文本，第一个参数为替换开始位置，第二个参数为需要被替换掉的长度，第三个参数为新加入的字符串。
  • subStringData()：用于获取子字符串，第一个参数为子字符串在Text节点中的开始位置，第二个参数为子字符串长度。

remove()

splitText() – 分割字符串

鼠标事件

click–单击
dblclick–双击
mousedown–按下触发。
mouseup–释放触发。
mousemove–内部移动触发
mouseenter–鼠标进入一个节点时触发，进入子节点不会触发这个事件
mouseout–鼠标离开一个节点时触发，离开父节点也会触发这个事件
mouseover–鼠标进入一个节点时触发，进入子节点会再一次触发这个事件
mouseleave–鼠标离开一个节点时触发，离开父节点不会触发这个事件
contextmenu–右键触发
wheel–滚轮触发

键盘事件

keydown

keypress

keyup

  • key：字符串，当前按下的键，默认为空字符串。
  • code：字符串，表示当前按下的键的字符串形式，默认为空字符串。
  • location：按下的键在键盘哪个区域
  • ctrlKey：布尔值，是否按下 Ctrl 键，默认为false。
  • shiftKey：布尔值，是否按下 Shift 键，默认为false。
  • altKey：布尔值，是否按下 Alt 键，默认为false。
  • metaKey：布尔值，是否按下 Meta 键，默认为false。
  • repeat：布尔值，是否重复按键，默认为false。

KeyboardEvent.getModifierState() – 是否按下指定功能键

焦点事件

focus – 元素节点获得焦点后触发，该事件不会冒泡。
blur – 元素节点失去焦点后触发，该事件不会冒泡。
focusin – 元素节点将要获得焦点时触发，发生在focus事件之前。该事件会冒泡。
focusout – 元素节点将要失去焦点时触发，发生在blur事件之前。该事件会冒泡。
